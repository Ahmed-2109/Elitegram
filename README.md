# Elitegram - Instagram Clone Webpage

Elitegram is a responsive Instagram clone webpage built using HTML, CSS, and JavaScript.  
The project recreates the layout and interactive experience of Instagram with modern UI elements, smooth animations, dark mode support, stories section, post interactions, modals, and responsive design.

---

## 🚀 Features

- Responsive Instagram-inspired UI
- Modern navigation bar
- Stories section with animations
- Like, comment, share, and save post actions
- Dark mode toggle
- Explore section
- Profile modal popup
- Story viewer modal
- Animated notifications
- Create post functionality
- Search bar interaction
- Double-click to like posts
- Smooth hover animations and transitions
- Mobile responsive layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons

## ⚡ Functionalities

### ❤️ Like System
Users can like and unlike posts with animated heart effects.

### 📌 Save Posts
Posts can be saved and unsaved dynamically.

### 👥 Follow Suggestions
Users can follow/unfollow suggested profiles.

### 🌙 Dark Mode
Theme switching between light and dark mode using local storage.

### 📷 Story Viewer
Interactive story viewer with auto progress animation.

### 📝 Create Post
Users can create posts with captions and image upload.

### 🔔 Notifications
Animated popup notifications for user interactions.
